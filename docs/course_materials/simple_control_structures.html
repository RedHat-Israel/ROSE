Blocks

  • All control structures (if, for / while, functions) content must be in a block.
  • Block are determined by indentation.
  • You can use either Tab or some spaces to indent.

if (4 < 5):
    print("Yes, It is true!")

Simple If

  • The general Python syntax for a simple if statement is:
    • 
if condition:
    doing-this-intended-block
    it-can-take-more-than-one-line
  • If the condition is true:do the intended block
  • If the condition is not true: skip the intended block
    • 
# What will be printed here?
age = 13
if age > 20:
    print('You are too old!')
print('You are young!')
                     
    
# And now?
age = 13
if age < 20:
    print('You are too old!')
    print('You are young!')

if-else Statements

  • What if the boolean expression is 'False' and not 'True'?
    • 
if condition:
  doing-this-block-if-condition-is-true
else:
  doing-this-block-if-condition-is-false
  • 'else' simply tells the computer what to do if the conditions of 'if' aren't met
    • 
# Example:

today = "Tuesday"
if today == "Sunday":
  print "Yay! Today we have ROSE!"
else:
  print "No ROSE for today :("

if-elif-else Statements

  • Sometimes we have to check multiple conditions.
    • 
if condition1:
    run-block-for-true-condition-1
elif condition2:
    run-block-for-true-condition-2
elif condition3:
    run-block-for-true-condition-3
else:
    run-block-for-all-conditions-were-false
  • 'elif' block actually combines 'else' and 'if' block.
  • This is what 'elif' uses for!

Quick tutorial

Installing

Tested only on Fedora

# dnf install fuse fuse-devel

# git clone https://github.com/nirs/slowfs.git

# cd slowfs

# pip install -r requirements.txt

Creating directories

Files under /realfs will be exposed
(slowly) under /slowfs

# mkdir /realfs /slowfs

How slow do you want to go today?

Create a configuration file

# cat slowfs.cfg
unlink = 60

Starting the file system

# python slowfs.py -c slowfs.cfg /realfs /slowfs

Testing locally

# touch /slowfs/test

# time rm /slowfs/test

real    1m0.063s
user    0m0.000s
sys     0m0.001s

It works!

How can we use this from another host?

Exporting via NFS

# cat /etc/exports
/slowfs    *(rw,sync,no_subtree_check,fsid=0)

Note: you must export the /slowfs directory, exporting the parent directory will not work

Restart NFS server

# systemctl restart nfs-server

Maybe there is a nicer way

Mounting on a client

# NFS 3
# mount my.server:/slowfs mountpoint

# NFS 4
# mount -t nfs4 my.server:/ mountpoint

Testing remotely

# touch mountpoint/test

# time rm mountpoint/test

real    1m0.063s
user    0m0.000s
sys     0m0.001s
