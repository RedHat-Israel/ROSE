Functions

Functions

  • Functions are chunks of code that tell python to do something
  • The reason to use functions is usability and code clearness
  • For complex programs, functions are essential

  • A function has:
    • A name
    • Parameters
    • A body

How to use


def say_hello(my_name):
    print(f"Hello {my_name}")

say_hello("Joe") # Hello Joe

Multiple Parameters

  • A function can have any number of parameters

def say_hello(first_name, last_name):
    print(f"Hello {first_name} {last_name}")

say_hello("Joe", "Cocker") # Hello Joe Cocker
  • Another way

first="Joe"
last="Cocker"
say_hello(first, last)

Variables Scope

  • When a function ends running, all its variables vanish

def scope():
    a=10
    print(a)

scope()    # 10
print(a)   #ERROR
  • If we have defined a variable in the program and the function redefined it, then the value in the function is used in the function scope

a=5
def scope():
    a=10
    print(a)

scope()    # 10
print(a)   # 5

The return statement

  • To let a function return a value, use the return statement

def my_function(x):
    return 5 * x

my_function(3)
print(my_function(5)) # 25
print(my_function(9)) # 45

Example - is even? 


def is_even(number):
    if number % 2 == 0:
        return True
    else:
        return False
                    

is_even(5) # False
is_even(3) # False
is_even(2) # True

Example - is prime?

  • A prime number is a natural number greater than 1 that is not a product of two smaller natural numbers.

def is_prime(x):
    if x < 2:
        return False
    for n in range(2, x - 1):
        if x % n == 0:
            return False
    return True
                  

is_prime(4) # False
is_prime(2) # True
is_prime(6) # False
is_prime(7) # True
is_prime(39) # False
is_prime(31) # True
