Variables

What is a Variable?

  • In computer programming, a variable has a name and contains a value.
  • The kind of value that a variable can hold is also called a data type

Variables Names

  • A variable can have a short name (like a and b) or a more descriptive name
    For example: age, carname, total_volume
  • Rules for Python variables:
    • A variable name must start with a letter or the underscore character
    • A variable name cannot start with a number
    • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
    • Variable names are case-sensitive (age, Age and AGE are three different variables)

Assignment

  • Variables in Python can hold values
  • This is done using the assignment operator (=)

a = 100
print(a) # 100

b = 200
print(b) # 200

c = a + b
print(c) # 300

d = c
print(d) # 300

Reassigning

  • You can modify the values in any variable by reassigning the value

a = 10
print(a)    # 10

a = 20
print(a)    # 20

a = 5.5
print(a)    # 5.5

a = "Hello"
print(a)    # Hello

Augmented Assignment

  • You can add to a variable using: a += 20
  • This the same as: a = a + 20

a = 50
print(a) # 50

a += 20
print(a) # 70
  • Other operations are also available:
  • -=, *=, /=, %= and **=

Name Errors

  • Getting the value of a variable that does not exist will produce an error
  • Each variable must be assigned before it is accessed

a = 50
print(a) # 50

print(b) # NameError: name 'b' is not defined

Multiple Assignments

  • Multiple assignments can be made in a single statement

a, b, c = 5, 3.2, "Hello"
  • Assigning the same value to multiple variables

x = y = z = 0

Numbers

Number data types

  • Python supports integers and floating point numbers
  • Integers and floating points are separated by the presence or absence of a decimal point:
    5 is an integer whereas 5.0 is a floating-point number

Type conversion

  • Operations between a float and an integer will result in a float
  • We can use functions like int() and float() to convert between types.
    These functions can convert also from strings

            >>> 1 + 2.0
            3.0
            >>> int(2.3)
            2
            >>> int(-2.8)
            -2
            >>> float(5)
            5.0
            >>> int("76")
            76

  • Same rules as in simple math including order of operations and parenthesis

a = (3 + 2)*5 + 6/3 #  5*5 + 2 =27

Arithmetic Operators

Operator Function Example
+ Add two operands or unary plus x + y
+2
- Subtract right operand from the left or unary minus x - y
-2
* Multiply two operands x * y
/ Divide left operand by the right one (always results into float) x / y
% Modulus - remainder of the division of left operand by the right x % y
(remainder of x/y)
** Exponent - left operand raised to the power of right x ** y
(x to the power y)

Examples


            x = 15
            y = 4

            print('x + y =',x+y) # Output: x + y = 19
            print('x - y =',x-y) # Output: x - y = 11

            print('x * y =',x*y) # Output: x * y = 60
            print('x / y =',x/y) # Output: x / y = 3.75

            print('x // y =',x//y) # Output: x // y = 3
            print('x % y =',x%y) # Output: x % y = 3

            print('x ** y =',x**y) # Output: x ** y = 50625

Booleans

Booleans

  • Python boolean types literals are True and False

      a = True
      if a:
         print("Hi there!")
      if not a:
         print("You shouldn't see me")

Booleans and other types

  • The following act as False:
    • None
    • 0
    • [] (or any empty sequence, strings included)
  • Everything else acts like True

Boolean Operations

Operation Result
x or y x = True, y = False, x or y -> True
x and y x = False, y = True, x and y -> False
not x x = False, not x -> True

Strings

What is a String?

  • A string is a data type used in programming, such as an integer and float
  • Represents text. For example, the word "hamburger" and the phrase "I ate 3 hamburgers" are both strings
  • Even "12345" could be considered as a string, if specified correctly.

Strings literals

  • The following strings literals are equivalent:

  "Hello World!"
  'Hello World!'
  """Hello World!"""
  '''Hello World!'''
  • Choose the quoting style that fits your need:

      "It's a very nice day."
      'The sign says "Hello World!".'

String literal escape sequences

  • \n :new line
  • \t :tab
  • \' and \" :quote and double quote
  • \\ :slash

      "I say \"Hello!\".\nYou say: \"Goodbye!\"."

Common String Methods

Checks

  • endswith, startswith

  s = 'hello world'
  s.startswith('he')  # -> True
  • isalnum, isalpha, isdigit, islower, isupper, isspace

    '123'.isdigit()  # -> True
    'Hello World'.islower()  # -> False

Searches

  • count

  s = 'hello world'
  s.count('l')  # -> 3
  • find

    s = 'hello world'
    s.find('l')  # -> 2
    s.find('t')  # -> -1

Manipulations

  • lower, upper , title , capitalize , swapcase

  s = 'hello world'
  s.title()  # -> 'Hello World'
  s.capitalize()  # -> 'Hello world'
  • replace

    s = 'hello world'
    s.replace('world', 'john')  # -> 'hello john'
  • strip, rstrip, lstrip - removes spaces and new lines from the ends of a string:

          s = '     hello!    \n'
          s.strip() # -> 'hello!'

+ and *


    'hello ' + 'world'  # -> 'hello world'

    'hello ' * 3  # -> 'hello hello hello '

String Slicing

Slicing

  • Index with square brackets: s[1]
  • Negative indexing gets elements from the end: s[-1]
  • Slicing
    • Basic syntax is s[start:end]: s[1:4]
    • Slicing from the beginning of the string: s[:5]
    • Slicing until the end of the string: s[3:]
    • Negative indexes can also be used in slices: s[-3:-1]

Examples


            s = 'Monty Python'

            # slicing (inclusive start, exclusive end)
            s[1:4]  # -> 'ont'

            # from start to index
            s[:4]  # -> 'Mont'

            # from index to end
            s[3:]  # -> 'ty Python'

Examples


            # can use negative indexes as well
            s[2:-2]  # -> 'nty Pyth'

            # jumps
            s[::2]  # -> 'MnyPto'
            s[1::2]  # -> 'ot yhn'

            # negative jumps
            s[::-1]  # -> 'nohtyP ytnoM'
