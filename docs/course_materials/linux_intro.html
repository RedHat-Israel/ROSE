Linux Introduction

Open a terminal window

  • Open the search menu:
    • Press the "Windows" key
    • Or move the mouse to the upper left corner of the screen
  • Type "terminal" in the search bar
  • Click on the icon named "Terminal"

$

pwd

  • Prints the current directory name

$pwd
/home/student
$

mkdir

  • Creates a directory in the current directory

$mkdir my_test
$
  • Creates a directory in a specific directory

$mkdir /home/student/my_test2
$
  • Creates a directory and all parent directories if missing

$mkdir -p my_test/dir2/dir3/dir4
$

cd

  • Change the working directory to another directory existing in the current directory

$cd my_test
$
  • Change to a specific directory (Full path)

$cd /home/student/my_test2
$

cd

  • Change to user home directory

  $cd
  $
          

  $cd ~
  $
  • Change to parent directory

  $cd ..
  $

ls

  • List current directory content

  $ls
  my_test
  my_test2
  $
  • List specific directory content

  $ls /home/student
  my_test
  my_test2
  $

ls -la

  • List current directory content with additional information and hidden files

  $ls -la
  total 12K
  drwxrwxr-x.  3 student student 4.0K Dec  9 22:57 .
  drwxrwxr-x. 10 student student 4.0K Dec  9 22:45 ..
  -rw-rw-r--.  1 student student    0 Dec  9 22:57 .bashrc
  drwxrwxr-x.  2 student student 4.0K Dec  9 22:45 my_test
  drwxrwxr-x.  2 student student 4.0K Dec  9 22:58 my_test2

  $

tree

  • List contents of directories in a tree-like format

$tree
.
├── my_test
│   └── dir2
│       └── dir3
│           └── dir4
└── my_test2

5 directories, 0 files
$

rmdir

  • Remove a directory from the current directory

$rmdir my_test
$
  • Remove a directory and all parent directories

$rmdir -p my_test/dir2/dir3/dir4
$

cat

  • Print the content of a file

$cat ~/.bashrc
# .bashrc

# Source global definitions
if [ -f /etc/bashrc ]; then
	. /etc/bashrc
fi
$

mv

  • Move a file to a different directory

$mv homework1.py /home/student/my_homeworks/
$
  • Change a file name

$mv homework1.py new_homework1.py
$

cp

  • Copy a file to a new file

$cp homework1.py homework1_backup.py
$
  • Copy a file to a different directory

$cp homework1.py /home/student/my_homeworks/
$
  • Copy a directory and its files into a different directory

$cp -r /home/student/my_homeworks/ backups/
$

rm

  • Remove a file

$rm homework1_backup.py
$
  • Remove a directory and all its content

$rm -r /home/student/backups/
$

touch

  • Create new files

$touch myfile1.txt myfile2.txt
$ls
myfile1.txt
myfile2.txt

head

  • Print the first part of files

$head -3 ~/.bashrc
# .bashrc

# Source global definitions
$

tail

  • Print the last part of files

$tail -3 ~/.bashrc
# export SYSTEMD_PAGER=

# User specific aliases and functions
$

diff

  • Compare files line by line

$diff myfile1.txt myfile2.txt
$

clear

  • Clear the terminal screen

$clear
$
