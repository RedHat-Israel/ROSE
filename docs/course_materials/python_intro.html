Python Introduction

What is Python?

  • A popular programming language.
  • It was written by Guido van Rossum in the 1990s.
  • It is used for:
    • Web development (server-side)
    • Software development
    • Mathematics
    • System scripting

Why Python?

  • Works on different platforms (Windows, Mac, Linux etc).
  • Python code is easy to read and understand.
  • Most programming tasks are easier with Python than with any other programming language.
  • The program runs immediately by Python's interpreter,
    so there's a fast feedback.

How to run Python?

  • Interactive mode:
    Open a terminal and type "python"
    • 
$python
Python 2.7.17 (default, Oct 21 2019, 17:20:57)
Type "help", "copyright", "credits" or "license" for more information.
>>>
    Use Ctrl+d for exit.

    Your code will be deleted once you close the session!

How to run Python?

  • Script mode:
    Use a file that contains Python code with .py extention

$python mycode.py
