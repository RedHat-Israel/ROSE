Python Introduction

What is Python?

  • A popular programming language
  • It was written by Guido van Rossum in the 1990s
  • It uses for:
    • Web development (server-side)
    • Software development
    • Mathematics
    • System scripting

Why Python?

  • Works on different platforms (Windows, Mac, Linux, etc.)
  • Python code is easy to read and understand
  • Most programming tasks are more accessible with Python than with any other programming language
  • The program runs immediately by Python's interpreter,
    so there's fast feedback

How to run Python?

  • Interactive mode:

How to run Python?

  • Script mode:
    Use .py extension for indicating the file contains Python code in it

                        $ python3 mycode.py

Comments in Python

  • A hash sign (#) begins a comment
  • All characters after the # and up to the end of the physical line are part of the comment
  • The Python interpreter ignores them

                        >>> # This is a comment
                        >>>
