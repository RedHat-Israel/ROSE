Modules and Packages

The 'import' statement

  • Allow use of other python files and libraries
  • Imports: import math
  • Named imports: import math as m
  • Specific imports: from math import pow
  • Import all: from math import * (dangerous! used only in very specific cases)

Example


import random
print random.randint(0, 5)

Modules


# utensils.py
def eat_soup():
    return 'spoon'

# main.py (option 1)
import utensils
print(utensils.eat_soup())

# main.py (option 2)
from utensils import eat_soup
print(eat_soup())

Packages

  • Packages are namespaces which contain multiple packages and modules themselves
  • Packages are simply directories, but there is a twist: each package/directory MUST contain a special file called __init__.py

Example


# fruits/__init__.py
# -- empty -- nothing here

# fruits/apple.py
def print_it():
    print('apple')

# main.py
from fruits import apple
apple.print_it()
