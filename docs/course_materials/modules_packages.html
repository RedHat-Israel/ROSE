Modules and Packages

The 'import' statement

  • Allow use of other python files and libraries
  • Imports: 
    • 
                        import math
  • Named imports: 
    • 
                          import math as m
  • Specific imports: 
    • 
                        from math import pow
  • Import all: (dangerous! used only in very specific cases)
    • 
                        from math import *

Example


import random
print (random.randint(0, 5))

Make a module

  • To make a module, create a Python file. Then import it like any other module. 

# utensils.py
def eat_soup():
    return 'spoon'
                    


# main.py (option 1)
import utensils
print(utensils.eat_soup())
                    


# main.py (option 2)
from utensils import eat_soup
print(eat_soup())

python module not found

  • If you get the error “ImportError: No module named “, this means the module is not installed or don't exsits.

Packages

  • Packages are namespaces which contain multiple packages and modules themselves
  • Packages are simply directories, but there is a twist: each package/directory MUST contain a special file called __init__.py

Example


# fruits/__init__.py
# -- empty -- nothing here

                    


# fruits/apple.py
def print_it():
    print('apple')
                    


# main.py
from fruits import apple
apple.print_it()
