Compound Data Types

List

Basics

  • An ordered collection of values
  • Index count starts from 0
  • Create an empty list

l = list()
type(l) # list
l = []
type(l) # list

Modifying

  • Appending values to the list:

l.append('apple')
print(l)             # ['apple']
l.append('orange')
print(l)             # ['apple', 'orange']
  • Creating a populated list:

l = ["orange", "apple", "strawberry", "banana"]
print(l)  # ["orange", "apple", "strawberry", "banana"]

Modifying

  • Adding on a specific index:

l.insert(3, 'melon')
print(l)
# ['orange', 'apple', 'strawberry', 'melon', 'banana']
  • Deleting an item on a specific index:

del l[0]
print(l)
# ['apple', 'strawberry', 'melon', 'banana']

Popping an element (remove it and return it's value):


print(l.pop())  # 'banana'
print(l)        # ['apple', 'strawberry', 'melon']
print(l.pop(0)) # 'apple'
print(l)        # ['strawberry', 'melon']

More actions


lst[i] = v      # Changes an element or slice by assigning
lst.remove(v)   # Removes a specific value
  • The .extend() method adds all the elements of an iterable to the end of the list, and returns None:

lst = [1,2,3]
lst.extend([4,5,6])
print(lst)  # [1,2,3,4,5,6]

Retrieving elements by index, negative index, and slicing:


print(l[0])     # 'orange'
print(l[8])     # IndexError: list index out of range
print(l[-1])    # 'banana'
print(l[1:3])   # ['apple', 'strawberry']
  • Lists can be concatenated with + and multiplied by *:

l = [25, 3, 14] + [5, 4] + [101, 2]
print(l)    # [25, 3, 14, 5, 4, 101, 2]

l = ["foo", "bar"] * 3
print(l)    # ['foo', 'bar', 'foo', 'bar', 'foo', 'bar']

In-place list operations

  • The .reverse() method changes a list in-place, and returns None:

fruit = ["orange", "apple", "strawberry"]
fruit.reverse()
print(fruit)  # ['strawberry', 'apple', 'orange']
  • Another handy way to reverse a list is by using a full slice with a negative step:

fruit = ["orange", "apple", "strawberry"]
fruit = fruit[::-1]
print(fruit)  # ['strawberry', 'apple', 'orange']
  • The .sort() method sorts elements in-place, and returns None:

fruit = ["orange", "apple", "strawberry"]
fruit.sort()
print(fruit)  # ['apple', 'orange', 'strawberry']

fruit.sort(reverse=True)
print(fruit)  # ['strawberry', 'orange', 'apple']

Tuple

Overview

  • Immutable: you cannot change its content
  • Tuples are used for grouping ordered data
  • Supports indexing and slicing syntax
  • Powerful assignment feature (packing/unpacking)
  • Creation is faster than a list

Creating a tuple


            t = tuple()
            type(t) # tuple

            t = (1, 2, 'hi')
            type(t) # tuple

            t = 1, 2, 'hi' # be careful
            type(t) # tuple

            t = tuple([1,2,'yo']) # casting a list to a tuple
            print(t) # (1, 2, 'yo')

Dictionary

Basics

  • Dictionary is an unordered set of pairs, where the keys are unique
  • dict() and {} are empty dictionaries
  • d[k] accesses the value mapped to k
  • d[k] = v updates the value mapped to k

Example


sports = {
    'Ralph' : 'Football',
    'Michael' : 'Basketball',
    'Shirly' : 'Baseball'
}
print(sports)
# {'Ralph': 'Football', 'Michael': 'Basketball',
#  'Shirly': 'Baseball'}

print(sports['Michael'])   # Basketball

Example


del sports['Michael']
print(sports)
# {'Ralph': 'Football', 'Shirly': 'Baseball'}

sports['Fred'] = 'Dancing'
print(sports)
# {'Ralph': 'Football', 'Shirly': 'Baseball',
#  'Fred': 'Dancing'}

sports['Ralph'] = 'Soccer'
print(sports)
# {'Ralph': 'Soccer', 'Shirly': 'Baseball',
#  'Fred': 'Dancing'}

Extras

Set

  • An unordered collection of unique elements (no duplicates)
  • Empty set: set()
  • {1, 'blah', 5, -1}

Set Methods


s.add(v) # adds a value to the set

s.remove(v) # removes v or KeyError if v not in the set.

s.discard(v) # removes v

s.difference(s2) or s - s2 # elements in s but not in s2

s.union(s2) or s | s2 # elements in s or s2

s.intersection(s2) or s & s2 # elements in both s and s2

s.update(s2) or s = s | s2 # updates s with s2 values

Lists, Dicts and references

  • In Python, Lists and Dicts are passed by reference to the object and not it's value.
  • For example, when using the assignment operator (=) on a list, a reference is created to the original list.
  • Which means that two variables point to the same value:

l = [5, 25, 100, 250, 1, 8]
print(l)    # [5, 25, 100, 250, 1, 8]
l2 = l
prtint(l2)  # [5, 25, 100, 250, 1, 8]

l.append("Hi!")
print(l)    # [5, 25, 100, 250, 1, 8, 'Hi!']
prtint(l2)  # [5, 25, 100, 250, 1, 8, 'Hi!']
  • l2 is not a copy of l, but a reference to the same list in python's memory.
  • To create a copy of the list, use the slicing operator ([:])

l3 = l[:]
l3.append("World")
print(l3)  # [5, 25, 100, 250, 1, 8, 'Hi!', 'World']
prtint(l)  # [5, 25, 100, 250, 1, 8, 'Hi!']
  • Same principle applies to dictionaries:

                        d1 = dict(eat='banana', drink='water') # {'eat': 'banana', 'drink': 'water'}
                        d2 = d1 # {'eat': 'banana', 'drink': 'water'}
                        d2['brush'] = 'teeth'
                        print(d1)
                        # {'eat': 'banana', 'drink': 'water', 'brush': 'teeth'}
                        print(d2)
                        # {'eat': 'banana', 'drink': 'water', 'brush': 'teeth'}
  • Dictionaries are being passed by reference and not by their raw value
  • There are two ways to duplicate a dictionary

                        d1 = {'eat': 'banana', 'drink': 'water'}
  • By creating a new dict out of an existing one:

                        d2 = dict(d1) # {'eat': 'banana', 'drink': 'water'}
  • Or by using the .copy() method:

                        d2 = d1.copy() # {'eat': 'banana', 'drink': 'water'}

Sequence operations

  • Check the length of lists, dicts, tuples and sets:

l = ['orange', 'apple', 'strawberry', 'banana']
print(len(l))     # 4
d = {'name': 'Tux', 'animal': 'penguin'}
print(len(d))     # 2
t = ('orange', 'apple')
print(len(t))     # 2
s = {'apple', 'banana', 'orange', 'apple'}
print(len(s))     # 3
  • For loops through the list:

for x in l:
    print(x)
# orange
# apple
# strawberry
# banana
