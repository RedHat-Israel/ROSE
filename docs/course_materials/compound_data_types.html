Compound Data Types

List

Basics

  • An ordered collection of values
  • Create an empty list

>>> l = list()
>>> type(l)
list
>>> l = []
>>> type(l)
list

Modifying

  • Appending values to the list:

l.append('apple')
print(l)             # ['apple']
l.append('orange')
print(l)             # ['apple', 'orange']
  • Creating a populated list:

l = ["orange", "apple", "strawberry", "banana"]
print(l)  # ["orange", "apple", "strawberry", "banana"]

Modifying

  • Adding on a specific index:

l.insert(3, 'melon')
print(l)
# ['orange', 'apple', 'strawberry', 'melon', 'banana']
  • Deleting an item on a specific index:

del l[0]
print(l)
# ['apple', 'strawberry', 'melon', 'banana']

Popping an element (remove it and return it's value):


print(l.pop())  # 'banana'
print(l)        # ['apple', 'strawberry', 'melon']
print(l.pop(0)) # 'apple'
print(l)        # ['strawberry', 'melon']

More actions

  • lst[i] = v: Change an element or slice by assigning to it
  • lst.extend(l): Add an iterable
  • lst.remove(v):remove a specific value

Sequence operations

  • All sequence operations are valid on lists:

l = ['orange', 'apple', 'strawberry', 'banana']
print(len(l))     #4
  • For loops through the list:

for x in l:
    print(x)
orange
apple
strawberry
banana

Retrieving elements by index, negative index, and slicing:


print(l[0])     # 'orange'
print(l[8])     # IndexError: list index out of range
print(l[-1])    # 'banana'
print(l[1:3])   # ['apple', 'strawberry']
l = [10,20,30,40]
print(l[::-1])  # [40, 30, 20, 10]
  • The in operator scans through all elements and return True or False:

if 'apple' in l:
    print("Yep")                # "Yep"
if 'basketball' in l:
    print("You won't see me")
  • Lists can be concatenated with + and multiplied by *:

l = [25,3,14] + [5,4] + [101, 2]
print(l)    # [25, 3, 14, 5, 4, 101, 2]

l = ["foo", "bar"] * 3
print(l)    # ['foo', 'bar', 'foo', 'bar', 'foo', 'bar']

In-place list operations

  • The .reverse() method changes a list in-place, and returns None:

fruit = ["orange", "apple", "strawberry"]
fruit.reverse()
print(fruit)  # ['strawberry', 'apple', 'orange']
  • The .sort() method sorts elements in-place, and returns None:

fruit = ["orange", "apple", "strawberry"]
fruit.sort()
print(fruit)  # ['apple', 'orange', 'strawberry']
fruit.sort(reverse=True)
print(fruit)  # ['strawberry', 'orange', 'apple']

Lists and references

  • When using the assignment operator (=) on a list, a reference is created to the original list:

l = [5,25,100,250,1,8]
print(l)    # [5, 25, 100, 250, 1, 8]
l2 = l
prtint(l2)  # [5, 25, 100, 250, 1, 8]
l.append("Hi!")
print(l)    # [5, 25, 100, 250, 1, 8, 'Hi!']
l2 = l
prtint(l2)  # [5, 25, 100, 250, 1, 8, 'Hi!']
  • l2 is not a copy of l, but a reference to the same list in python's memory.
  • To create a copy of the list, use the slicing operator ([:]) ::

l3 = l[:]
l3.append("World")
print(l3)    # [5, 25, 100, 250, 1, 8, 'Hi!', 'World']
prtint(l)  # [5, 25, 100, 250, 1, 8, 'Hi!']

Tuple

Overview

  • Immutable: you cannot change its content
  • Tuples are used for grouping ordered data
  • Support indexing and slicing syntax
  • Powerful assignment feature (packing/unpacking)
  • Creation is faster than a list

Creating a tuple


            >>> t = tuple()
            >>> type(t)
            tuple

            >>> t = (1, 2, 'hi')
            >>> type(t)
            tuple

            >>> t = 1, 2, 'hi' # be careful
            >>> type(t)
            tuple

            >>> t = tuple([1,2,'yo'])
            >>> t
            (1, 2, 'yo')

Dictionary

Basics

  • Dictionary is an unordered set of key: value pairs, where the keys are unique
  • dict() and {} are empty dictionaries
  • d[k] accesses the value mapped to k
  • d[k] = v updates the value mapped to k

Example


sports = {'Ralph' : 'Football',
          'Michael' : 'Basketball',
          'Shirly' : 'Baseball'}
print(sports)
# {'Michael': 'Basketball',
#  'Ralph': 'Football', 'Shirly': 'Baseball'}
print(sports['Michael'])   # Basketball

Example


del sports['Michael']
print(sports)
# {'Ralph': 'Football', 'Shirly': 'Baseball'}
sports['Fred'] = 'Dancing'
print(sports)
# {'Fred': 'Dancing',
#  'Ralph': 'Football', 'Shirly': 'Baseball'}
sports['Ralph'] = 'Soccer'
print(sports)
# {'Fred': 'Dancing',
#  'Ralph': 'Soccer', 'Shirly': 'Baseball'}

Set

Basics

  • List with no order, no duplicates
  • Empty set: set()
  • {1, 'blah', 5, -1}

Methods

  • s.add(v): adds a value to set
  • s.remove(v): removes v
  • s.discard(v): removes v
  • s.difference(s2) or s - s2: elements in s but not s2
  • s.union(s2) or s | s2: elements in s or s2
  • s.intersection(s2) or s & s2: elements in s and s2
  • s.update(s2) or s = s | s2: updates s with s2 values
